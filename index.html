Network Configuration Guide

Email Server Topology: Branch-1 | Head-Office | Branch-2

Network Topology & Components

  • Router: 3x Cisco 2911
  • Switch: 3x Cisco 2960
  • End Devices: 6x Generic PCs
  • Server Devices: 3x Server
  • Cabling: Copper Straight-Through & Serial Cables

Step 1: Router IP Address Configuration

Assign IPs to the physical interfaces of the routers as per the topology diagram.

Branch-Office-1

                Router(config)#hostname BR-1
                BR-1(config)#int g0/1
                BR-1(config-if)#ip add 10.10.10.1 255.255.255.0
                BR-1(config-if)#no sh
                BR-1(config-if)#int g0/0
                BR-1(config-if)#ip add 20.20.20.1 255.255.255.0
                BR-1(config-if)#exit
                BR-1(config)#ip dhcp pool BR-1-Lan
                BR-1(dhcp-config)#network 10.10.10.0 255.255.255.0
                BR-1(dhcp-config)#default-router 10.10.10.1
                BR-1(dhcp-config)#dns-server 10.10.10.10

Head-Office

                Router(config)#hostname Head-Office
                Head-Office(config)#int g0/0
                Head-Office(config-if)#ip add 20.20.20.2 255.255.255.0

                Head-Office(config-if)#hostname Head-Office

Step 2: Dynamic Routing (RIP)

To ensure all networks can communicate, enable RIP version 2 on all routers.

BR-1

                BR-1(config)#router rip 
                BR-1(config-router)#version 2
                BR-1(config-router)#no auto-summary 
                BR-1(config-router)#network 10.10.10.0
                BR-1(config-router)#network 20.20.20.0
                BR-1(config-router)#network 30.30.30.0
                BR-1(config-router)#network 40.40.40.0
                BR-1(config-router)#network 50.50.50.0

Head-Office

                Head-Office(config)#router rip 
                Head-Office(config-router)#version 2
                Head-Office(config-router)#no auto-summary 
                Head-Office(config-router)#network 10.10.10.0
                Head-Office(config-router)#network 20.20.20.0
                Head-Office(config-router)#network 30.30.30.0
                Head-Office(config-router)#network 40.40.40.0
                Head-Office(config-router)#network 50.50.50.0

BR-2

                BR-2(config)#router rip 
                BR-2(config-router)#version 2
                BR-2(config-router)#no auto-summary 
                BR-2(config-router)#network 10.10.10.0
                BR-2(config-router)#network 20.20.20.0
                BR-2(config-router)#network 30.30.30.0
                BR-2(config-router)#network 40.40.40.0
                BR-2(config-router)#network 50.50.50.0

Note: Apply this to all three routers so they share their routing tables.
